## 2. 页面/组件清单

### 2.1 启动流程

```
main.py
  │
  ├── SplashScreen (启动画面)
  │     └── 淡入淡出动画 (750ms 淡入 + 500ms 显示 + 750ms 淡出)
  │
  └── Discover_gui.run_gui()
        │
        ├── QApplication 初始化
        ├── DiscoverApp (加载配置)
        ├── preload_next_batch (预加载歌曲)
        ├── create_tray_icon (创建托盘)
        └── app.exec() (事件循环)
```

### 2.2 核心组件

| 组件 | 文件 | 描述 |
|------|------|------|
| `SplashScreen` | main.py | 启动画面，展示项目 Logo 并淡入淡出 |
| `DiscoverOverlay` | Discover_gui.py | 主浮窗界面，全屏透明显示歌曲卡片 |
| `SongCardWidget` | Discover_gui.py | 歌曲卡片组件，展示歌曲信息 |
| `SettingsWindow` | settings/setting_gui.py | 设置窗口，模态对话框 |
| 托盘菜单 | Discover_gui.py | 系统托盘右键菜单（发现/设置/快捷键/重启/退出） |
| 各平台 run.py | platforms/{Platform}/run.py | 各平台播放函数（scheme URL 唤起 + 窗口最小化） |

### 2.3 页面跳转

| 从 | 触发方式 | 到 |
|---|---------|---|
| 托盘图标 | 双击 / 右键菜单"发现歌曲" | DiscoverOverlay |
| 托盘图标 | 右键菜单"设置" | SettingsWindow |
| 托盘图标 | 右键菜单"退出" | 退出应用 |
| DiscoverOverlay | 点击关闭按钮 | 隐藏浮窗 |
| SettingsWindow | 点击"保存" | 保存并关闭 |
| SettingsWindow | 点击"取消" | 关闭不保存 |

## 3. 关键交互

### 3.1 发现歌曲流程

1. 用户点击托盘图标或按快捷键
2. 显示 `DiscoverOverlay` 全屏浮窗
3. 加载已启用歌单中的歌曲（支持网易云/QQ音乐/酷狗音乐/Spotify）
4. 以卡片形式展示（每行 5 张卡片）
5. 用户点击歌曲卡片 → 调用平台对应 run.py 播放歌曲
6. 用户点击关闭按钮 → 隐藏浮窗
7. 预加载下一批歌曲到缓存

### 3.2 设置修改流程

1. 用户打开设置窗口
2. 修改各项设置
3. 点击"保存" → 写入配置文件
4. 如有需要，刷新相关组件

## 4. 数据流

```
用户操作 → Discover_gui / SettingsWindow
    │
    ▼
PASetting / GuiSetting (数据模型)
    │
    ▼
settings/music_setting.py / settings/gui_setting.py
    │
    ▼
user_data/settings/*.json (持久化)

platforms/ 各平台歌单/卡片/播放
    ├── NeteaseCloudMusic/
    ├── QQMusic/
    ├── KugouMusic/
    └── Spotify/
```
